para.cpp  10.04.02 -  решение да се изпрати до 17.04.02

В равнината са дадени са N (1<N<30) правоъгълника, чиито страни са 
успоредни на координатните оси Ox и Oy. Разстояние между два 
правоъгълника се нарича дължината на най-късата отсечка, която 
свързва точка от единия правоъгълник с точка от другия. Ако двата 
правоъгълника имат поне една обща точка, тогава разстоянието между 
тях е нула. Намерете разстоянието между двата най-близки и 
двата най-далечни правоъгълници.

Вход - файл para.in:
В първия ред е записано числото N, а във всеки от 
следващите редове са дадени по 4 цели числа между -10000 и
10000, разделени с шпации. Първите две числа означават x- и 
y-координатите на горния ляв ъгъл на поредния даден правоъгълник, 
а вторите две числа са x- и y-координатите на долния десен ъгъл 
на същия правоъгълник. Файлът съдържа много примери, наредени 
един след друг.

Изход - файл para.out
За всеки пример файлът съдържа ред с две числа - разстоянията 
между двата най-близки и двата най-далечни правоъгълници, разделени 
с един интервал. Числата да са закръглени (по общоприетите правила за
закръгляване) с 3 значещи цифри след десетичната точка.

Примерен вход:
3
0 1 1 0
30 40 40 30
-1 0 0 -2
2
1 1 101 0
-101 1 -1 0 

Примерен изход:
0 41.426
2 2

"Числата да са закръглени (по общоприетите правила за
закръгляване) с 3 значещи цифри след десетичната точка."

#include <iostream>
using namespace std;
int main()
{
 double x[8]={2, 2.1, 2.12, 2.123, 2.1234, 2.1238, 2.999, 2.9999};
 for (int i=0; i<8; i++)
 cout << static_cast<int>((x[i]+0.0005)*1000)/1000.0 << "\n";
 cin.get();
 return 0;
}
