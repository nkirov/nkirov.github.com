Comprehensive TeX Archive Network (CTAN)

The file CTAN.sites contains inportant information if you wish to load files directly from CTAN sites. Alternatively, you may wish to obtain a copy of the CTAN via CD-ROM (see help/CTAN.cdrom for details). There is a copy of CTAN distribution via CD-ROM (in German) which is avaiable at the Institute of Mathematics and Informatics. Here can be found the contents of the first CD-ROM [563K] and the second CD-ROM [251K].

The TeX and LaTeX Catalogue: This is a catalogue of TeX and LaTeX related packages and tools, mostly available on CTAN, the Comprehensive TeX Archive Network. Most entries have a link to a location on the local CTAN directory hierarchy which takes you to the source directory for that entry.
