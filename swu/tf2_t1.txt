< Това са въпросите от теста, като са дадени по два отговора -
  един верен и един грешен.>


1. Отбележете правилните дефиниции на променливи:
+ var k: integer;
- var kl; mn: integer;

2. Дадени са следните дефиниции:
var m, n: integer;
    b1, b2: boolean;
    by, bye: byte;
Отбележете верните аритметични изрази и оператори за присвояване.
+ m:=100;
- b1:=10;

3. Кои от следните оператори за цикъл,
включени в програма, ще произведат безкраен цикъл.
Променливата i е от тип integer и има стойност 0.
+ while true do i:=i-1;
- for i:=0 to 10 do writeln(i);

4. Дадени са следните дефиниции:
const k=0;  e=1;   m=2;
Отбележете изразите, чиято стойност е true.
+ k>=0
- k+m<>2*e

5. Отбележете валидните имена на променливи.
+ AlKapone
- bye-bye

6. Даденa е следната част от програма:
var age1, age2, k: integer;
begin
   age1:=21;
   age2:=81;
   k:=9;
Отбележете изразите, които имат стойност 9.
+ age2 div k
- (age2-2) div k

7. Отбележете правилните дефиниции на константи:
+ const k=10;
- type t='X';

8. Отбележете правилните дефиниции на типове данни:
+ type Etype=1..10;
- const Ed=1;

9. Отбележете израза, който след като се постави на мястото на ...,
програмата отпечатва числото 15:
const k: integer=15;
begin
   if (...) then writeln(k);
end.
+ k=15
- k<>15

10. Отбележете израза, който след като се постави на мястото на ...,
програмата отпечатва 3 пъти буква A:
var i, k: integer;
for i:=4 downto 1 do
begin
   ...
   if i>=k then write('A');
end.
+ k:=2;
- k:=i-1;

11. Отбележете вярно написаните оператори за въвеждане и извеждане
на данни, ако са зададени следните променливи:
const up: real=1.52;
      ix: integer=73;
+ readln(up);
- writeln(ix:5:2);

12. Дадени са следните дефиниции:
var n : integer;
    b : boolean;
    ra, rb: real;
    c1, c2: char;
Отбележете верните аритметични изрази и оператори за присвояване.
+ n:=100;
- b:=10;
