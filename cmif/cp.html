The problems - arcive_text.ZIP or abcd_tex.ZIP
The archive arcive_text.ZIP contains the file ABCD.PS (problems A, B, C and D ) and the files PRIME.RTF, LINES.RTF, PIC.RTF and PASS.RTF. abcd_tex.ZIP contains A-D in LaTeX format.
 
Problem Name Name of problem files Number of teams, which are solved the problem
A The Shapley-Shubik index of power shsh 3 shsh.pdf
B Period period 13 period.pdf
C Groups of points points 8 points.pdf
D Happy numbers happy 18 happy.pdf
E Prime numbers prime 0 prime.html
F Bus lines lines 0 lines.html
G Picture pic 0 pic.html
H Passwords pass 5 pass.html

Test examples - archive_inp.ZIP, archive_inp_G.ZIP
Problem H has 12 test examples, contain in different files. Other problems has only one file, containing all test examples The test examples of problem G are 24 MB and are given separately.

Solutions - arcive_source.ZIP
ZIP-files contains 20 directories - one for each team. The team directory contains the final program sources, given the jury in SUBMIT procedure.
