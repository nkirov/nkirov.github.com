NETB131 Programming Project [2016/2017]

You have to write a short review of a programming language. You can find your individual task in List of students. For obtaining information use Google search and Wiki as sources of data and links to other sites. Prepare the text in html file format. Recommended html editor - Mozilla Software SeaMonkey.

Remark. You must write code for the task: Input an integer number n and output the sum: 1+22+32+...+n2. Use input validation for n to be positive.

Send me the file by e-mail (nkirov<at>nbu.bg). The answer may be OK or it may contain some recommendations for improving the project. After confirmation (OK), print out (on paper) the text and leave the printout in office 611-2.
