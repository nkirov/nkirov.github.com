    Задача 2. Щастливи двоични числа

    Щастливо двоично число ще наричаме естествено число, чието представяне в двоична бройна система съдържа еднакъв брой единици и нули. Например числото 910 = 10012 е щастливо двоично число, а 2210 = 101102  - не е. Да се  напише програма happy.exe, която да намира броят на щастливите двоични числа в зададен интервал.

    ВХОД - файл happy.inp
    Файлът съдържа няколко реда, като на всеки ред са записани 2 числа (не по-големи от 109) - граници на интервала, в който се търси броят на щастивите двоични числа. Двойката числа 0, 0 задава края на входния файл.

    ИЗХОД - файл happy.out
    За всеки интервал от входния файл да се отпечата на отделен ред броят на щастивите двоични числа в този интервал.

    Пример.
30 39
256 512
0 0

    Решение на примера.
3
0
