NETB330 Advanced Programming Project - 1 [2014/2015]

Textbook: Steven S. Skiena,  Miguel A. Revilla,  Programming Challenges, The Programming Contest Training Manual Series: Texts in Computer Science, Springer, 2003 [Call Number: Ð§ 681.325.3 / S 66][Moodle]

You have to complete the following algorithm:
First step:
  1. Register yourself as a student at

    Online Judge

  2. Choose a problem to solve among these, which are free yet from:
    • Root :: Programming Challenges (Skiena & Revilla) :: Chapter 2)
    • Root :: Programming Challenges (Skiena & Revilla) :: Chapter 3)
  3. E-mail me your choice.
Second step, when you are assigned the Problem No.
  1. Read Chapter 1, Chapter 2 and Chapter 3 from the textbook.
  2. Write a program in C++ for solving your problem.
  3. Check the program using SUBMIT 
  4. If  Solved appears in the column Status and in the row of your task, E-mail the source code to nkirov<at>nbu.bg and stop.
  5. Else go to 4.
