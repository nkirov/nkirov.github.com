Projects will be given in December!

    Remark. You must write code for the task: Input an integer number n and output the sum: 1+22+32+...+n2. Use input validation for n to be positive.
    Send me the file by e-mail (nkirov<at>nbu.bg). The answer may be OK or it may contain some recommendations for improving the project. After confirmation (OK), print out (on paper) the text and give the paper to me.
