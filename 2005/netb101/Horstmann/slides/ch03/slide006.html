Constructing Objects (Time class)

  • To use the Time class we must include the file that contains its definition:
  • #include "ccc_time.cpp"
    • We use " " instead of < > because the file is not a standard header.
    • The name stands for Computing Concepts with C++ Essentials.
    • Usually we don't use #include with .cpp files (see Chapter 6).
