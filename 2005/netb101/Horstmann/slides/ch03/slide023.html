Choosing a Coordinate System

  • The default coordinate system for the book's graphics classes:
    • The origin (0, 0) is at the center.
    • x- and y- axis range from -10.0 to 10.0
  • The user can reset the coordinate system: 
    cwin.coord(x_left, y_top, x_right, y_bottom)
  • Example:

    For the following data, the x-axis should range from 1 to 12, and the y-axis should range from 11 to 33:

    cwin.coord(1, 11, 12, 33);
