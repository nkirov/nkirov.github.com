Constructing Objects

  • An object is a value that can be created, stored and manipulated.
  • Every object in C++ must belong to a class.
  • A class is a data type (like int or double) that is programmer defined.
  • The text contains a Time, Employee, and 4 shape classes for us to learn to work with classes.
    • Remember, these are programmer defined and not part of standard C++.
