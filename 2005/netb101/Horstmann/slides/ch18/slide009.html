Adding a Text Control to the Frame

  • We have to make a slight change to the application class to show a TextFrame, not a wxFrame. 
    class TextApp : public wxApp
{
public:
   TextApp();
   virtual bool OnInit();
private:
   TextFrame* frame;
};

TextApp::TextApp()
{
   frame = new TextFrame();
}
