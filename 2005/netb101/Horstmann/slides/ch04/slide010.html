Relational Operators

  • C++ has six relational operators to implement conditions:
    C++
    Description
    Example
    Notes
    >
    greater than
    a > 5
    >=
    greater than or equal to
    x >= 5
    		 Be careful to not write =>
    Remember that the symbols appear in the order you say them
    <
    less than
    x < 10
    <=
    less than or equal to
    x <= 100
    		 Be careful to not write =<
    Remember that the symbols appear in the order you say them
    ==
    equal to
    a == 5
    		 Don't confuse with =
    which is assignment.
    !=
    not equal
    a != 5
    		 The ! is supposed to be the line that "crosses through" the equal sign.
