NETB131 Programming Project [2013/2014]

You have to write a short review of a programming language. You can find your individual task in List of students (to be available after the second test). To collect the data use WWW, especially Google for searching and Wikipedia for a source and links container. Prepare the text in html file format. Recommended html editor - Mozilla Software SeaMonkey.
    Send me the file by e-mail (nkirov<at>nbu.bg). The answer may be OK or it may contain some recommendations for improving the project. After confirmation (OK), print out (on paper) the text and give it to me.
