Посочете верните оператори.
Променливата е i от тип int, а променливата x -- от тип double.

+ x=i++;
- i=+-2;



При кои израз, поставен на мястото на ..., частта от програма:
   int k = 15;
   if (...) cout << k << "\n";
ще отпечати числото 15.

+ k==15
- k<15



Отбележете инициализациите, които, поставени на мястото на ...
в частта от програма:
   int ...;
   if (n>12) cout << n   << "\n";
   else      cout << n+1 << "\n";
на екрана ще се изведе четно число.

- n=0
+ n=1



Отбележете логическите изрази, в които са възможни т.н.
отложени пресмятания. Променливите a, x са от тип double,
променливата s -- от тип string и променливите
flag1, flag2 -- от тип bool.

+ cin.fail() or (a<0)
- not(x>=0)



Дадени са следните дефиниции на променливи:
   bool zero=false, one=true;
Отбележете изразите със стойност true.

+ one && true
- zero || false



Отбележете верните твърдения за функции.

+ Входна стойност на функцията може да бъде константа.
- Формалните параметри се задават при извикването на функцията.



Дадена е следната декларация на функция:
   int test(int n, int &m);
Отбележете верните оператори за извикване на функцията test.
Дадено е още int ind = 0;.

+ cout << test(1, ind);
- cin >> test(0, ind);



Дадена е функцията:
   int fun(int n)
   {
    if (n%3 > 1) return n;
    else         return n+1;
   }
Отбележете случаите, когато ще се отпечата четно число.

- cout << fun(0);
+ cout << fun(1);



Отбележете оператор от главна функция main, който ще отпечата числото 24.
   long new_fact(int n)
   {
    if (n == 1) return 1;
    else
    {
     long result = n * new_fact(n - 1);
     return result;
    }
   }

+ cout << new_fact(4);
- cout << new_fact(24)/24;
